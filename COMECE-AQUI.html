🎯 Como Começar

1

Abra o Arquivo index.html

Localize a pasta website-puro no seu computador e clique duplo no arquivo index.html

Caminho completo: website-puro/index.html

2

Pronto! O Website Rodará

O seu navegador padrão abrirá com o website completo. Nenhum comando no terminal é necessário!

👉 O site rodará localmente sem precisar de servidor

3

(Opcional) Use Live Server no VS Code

Para melhor experiência de desenvolvimento:

  1. Abra a pasta website-puro no VS Code
  2. Instale a extensão "Live Server" (se não tiver)
  3. Clique direito em index.html → "Open with Live Server"
